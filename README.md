# singleDiv
写这个东西的起因是因为群里的一个朋友说到了li里面在头部加图片无法实现垂直居中，<br>
我们讨论觉得利用伪元素的方式，（利用::before和::after的形式），来实现居中。<br>
li:before {
  content:"url(#)",
  display:block;
  vetical-align:middle;
}<br>
即可实现在li里面居中。<br>
所以我们又想到用一个div实现做一个简单的东西--精灵球<br>
